Rain

Type Creative Coding
Category Particle System
Year 2024

Overview

Rain is an interactive particle system visualization built with p5.js. It creates a mesmerizing rain effect where animated droplets fall from the top of the screen and react to your mouse position.

The unique feature of this project is the mouse interaction - raindrops are repelled away from your cursor, creating an interactive umbrella-like effect as you move your mouse around the canvas.

How It Works

The visualization uses object-oriented programming with a Rain class that manages individual droplets:

  • Each raindrop has its own position, velocity, length, and color
  • 10 new raindrops are spawned every frame
  • Droplets continuously fall and update their positions
  • Mouse proximity detection repels nearby drops
  • Renders on a 600x400 canvas

Technologies Used

This creative coding project demonstrates:

  • p5.js - Creative coding framework
  • Object-Oriented JavaScript - Rain class with properties and methods
  • Particle systems - Managing many independent objects
  • Mouse interaction - Real-time cursor tracking and collision detection
