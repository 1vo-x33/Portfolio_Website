Raad Het Nummer

Type Web Game
Category Interactive
Year 2024

Overview

"Raad Het Nummer" (Guess the Number) is a classic number guessing game built with vanilla JavaScript. The game challenges players to guess a randomly generated number between 0 and 100.

With each guess, the game provides helpful hints telling the player if their guess was too high ("Te hoog!") or too low ("Te laag!"), guiding them toward the correct answer.

How It Works

The game mechanics are simple but engaging:

  • A random number between 0-100 is generated when the game starts
  • Players enter their guess in the input field
  • Feedback is given: "Te laag!" (too low), "Te hoog!" (too high), or "Juist!" (correct)
  • The number of attempts is tracked
  • A reset button allows starting a new game

Technologies Used

This project demonstrates fundamental web development skills:

  • HTML - Form input and button elements
  • CSS - Fixed positioning layout with styled buttons
  • JavaScript - Random number generation, comparison logic, and DOM updates
