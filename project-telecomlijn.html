Telecomlijn Website

Type Business Website
Category Web Development
Year 2025

Overview

Telecomlijn Zakelijk is a professional B2B telecommunications website I built during my internship. The site serves as the complete online presence for a Dutch telecommunications company, featuring multiple service pages, live chat integration, remote support functionality, and interactive UI elements.

Built with Next.js 16 and React 19, the website includes 11+ service detail pages, a comprehensive contact system, and real-time customer support integrations. I implemented custom animations, form validation, and a component library with 40+ reusable UI components.

3CX Live Chat Integration

Integrated a live chat widget connected to the 3CX Phone System, allowing real-time customer communication directly on the website. The implementation includes:

  • Dynamic script loading from 3CX cloud services
  • Custom HTML element with TypeScript declarations
  • Error handling and cleanup on component unmount
  • Custom styling to match the site's design

AnyDesk Remote Support

Built a dedicated remote support page with multi-platform download options for the company's custom AnyDesk client:

  • Windows (.exe) and macOS (.dmg) desktop downloads
  • Android APK and iOS App Store links
  • Step-by-step guide with 4 instructional steps
  • Security information and use cases section

Interactive Features

The website includes various interactive elements and animations:

  • Animated service icon orbit on the homepage with rotating SVG circles
  • Service filtering system with 5 categories and grid/list view toggle
  • Contact form with React Hook Form and Zod validation
  • Character counter with live feedback (max 600 chars)
  • FAQ accordions with expandable sections
  • Custom float and pulse-glow CSS animations
  • Hover effects with shadow, translate, and scale transitions

Pages Built

I developed multiple pages for the website:

  • Homepage with hero section, stats, services, and partner logos
  • Services hub with 22 services across 4 categories
  • 11 individual service detail pages (VoIP, 3CX, Security, Fiber, etc.)
  • Contact page with form and WhatsApp integration
  • Support hub with FAQ and status monitoring
  • Remote support page with downloads
  • About and Partners pages
  • Phone numbers page with pricing info

Technologies Used

This project leverages a modern tech stack:

  • Next.js 16 with App Router - Server-side rendering and routing
  • React 19 - Latest React with concurrent features
  • TypeScript 5 - Type-safe development
  • Tailwind CSS 4 - Utility-first styling with custom animations
  • Radix UI - 40+ accessible component primitives
  • React Hook Form + Zod - Form handling and validation
  • Vercel Analytics - Performance monitoring
  • Lucide React - Icon library
